How many variables named i?

Python and Go 1.21 answer: one. Its value is 0, then 1, then 2, then 3. Loop variables are per-loop, not per-iteration. closures' closures all see that one variable. When the closures are called, its value is 3.

Javascript and Go 1.22 answer: three (or is it four?). The first has value 0, the second 1, the third 2, and their values don't change. Loop variables are per-iteration, not per-loop. closures' closures see the corresponding variable.
